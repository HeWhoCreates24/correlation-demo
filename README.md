## Correlation Analysis

This repository contains a dataset and a Python script to perform correlation analysis on various variables related to inventory management.

### Dataset

The dataset is stored in a CSV file named `correlation.csv` and contains the following variables:

- Supplier Lead Time
- Inventory Levels
- Order Frequency
- Delivery Performance
- Cost Per Unit

### Python Script

The Python script `correlation_analysis.py` performs the following tasks:

1. Reads the dataset from the CSV file
2. Calculates the correlation matrix for the dataset
3. Prints the correlation matrix to the console
4. Saves the correlation matrix to a new CSV file named `correlation_matrix.csv`

### Usage

To run the Python script, use the following command:

```bash
python correlation_analysis.py
```

The script will generate a correlation matrix and save it to a new CSV file named `correlation_matrix.csv`.
